## Steps to setup PKS API
Navigate to `/api/PKS`
```root@photon-machine [ /usr/local ]# cd vmw-e2e/api/PKS/```

Install **unzip**
```root@photon-machine [ /usr/local/vmw-e2e/api/PKS ]# yum install unzip```
